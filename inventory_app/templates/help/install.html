ติดตั้ง Sendy ลงหน้าจอโฮมเพื่อเปิดแอปเร็วขึ้น เหมือนแอปปกติ ไม่ต้องพิมพ์ URL ใหม่ทุกครั้ง

{# ── Android Chrome ── #}
Android — Chrome
  1. เปิด Sendy ใน Chrome บน Android
  2. แตะเมนู ⋮ (มุมบนขวา)
  3. เลือก "ติดตั้งแอป" หรือ "เพิ่มไปยังหน้าจอโฮม"
  4. แตะ ติดตั้ง เพื่อยืนยัน
  5. ไอคอน Sendy จะปรากฏบนหน้าจอโฮมทันที
{# ── iPhone Safari ── #}
iPhone — Safari
  1. เปิด Sendy ใน Safari บน iPhone (ต้องใช้ Safari เท่านั้น)
  2. แตะปุ่ม แชร์ (กลางแถบด้านล่าง)
  3. เลื่อนลงแล้วเลือก "เพิ่มไปยังหน้าจอโฮม"
  4. แก้ชื่อ (ถ้าต้องการ) แล้วแตะ เพิ่ม
  5. ไอคอน Sendy จะปรากฏบนหน้าจอโฮม — เปิดแบบ Standalone ไม่มี address bar
iPhone อาจขอ Login ใหม่ครั้งแรกหลังติดตั้ง เป็นเรื่องปกติ เกิดจาก cookie sandbox ของ Safari
